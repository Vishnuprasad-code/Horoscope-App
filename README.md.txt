A python webapp to display user's horoscope using Flask, HTML, CSS, BeautifulSoup

*The app will ask for user's email-id and zodiac sign. 
*Then depending on the zodiac sign corresponding horoscope is extracted from 
	atrology.com using beautifulsoup.

*And the same is sent to the user's email address. 

*Furthermore, the user can see the horoscopes of the previous day 
	and next day in the browser once submit is pressed...
